# ExitDebt

**See your debt clearly. Solve it smartly.**

ExitDebt is a full-spectrum debt platform for salaried Indians. Users enter their PAN and phone number, get an instant CIBIL-powered debt health score with unique intelligence tools, and get 3 months of free access ‚Äî all in 30 seconds.

```
PAN + Phone  ‚Üí  CIBIL Pull  ‚Üí  Full Dashboard (3mo free)  ‚Üí  Lite/Shield Subscribe
```

1. **Enter PAN + Phone** ‚Äî Verified via OTP
2. **Instant CIBIL Pull** ‚Äî All loans and credit cards auto-populated
